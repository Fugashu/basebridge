# BaseBridge.app

Open Source, feeless GUI implementation for bridging ETH from the Ethereum Mainnet to the Base Mainnet.

## Usage

- Connect your wallet
- Enter the desired amount in ETH you want bridged
- Click 'Deposit'
- Confirm the transaction in Metamask
- Done
